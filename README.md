# SentinelNet - Bittensor Subnet Ideathon Submission

Cybersecurity has one property that most AI benchmarks lack: a binary ground truth. A vulnerability either exists or it does not. An exploit either fires or it fails. There is no interpretation, no partial credit, no room for ambiguity.

SentinelNet is built on that insight. Miners analyse network logs, smart contract bytecode, and system snapshots to identify real threats. Validators confirm every finding the only way that matters — by attempting the exploit in an isolated sandbox. If it fires, intelligence is proven. If it does not, the claim is rejected. Every outcome is recorded on-chain, auditable by anyone, owned by no one.

## Overview

SentinelNet is a decentralized cybersecurity intelligence network built on Bittensor. It leverages a dual-role architecture where miners identify threats and validators verify them through sandboxed exploit verification.
